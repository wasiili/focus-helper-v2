# Focus Helper 专注助手 - 产品设计文档

## 1. 产品概述

### 产品名称
Focus Helper (专注助手)

### 产品类型
鸿蒙原生应用 (HarmonyOS)

### 核心功能
帮助用户进入专注模式，记录专注时长，提供轻音乐播放，完成后给予正向反馈

### 目标用户
- 学生：学习时需要专注
- 工作者：需要高效工作时间
- 任何人：想要培养专注习惯

---

## 2. 功能列表

### F1: 专注模式入口
- 用户点击"开始专注"按钮进入专注模式
- 显示当前日期和时间

### F2: 目标设置
- 进入前设置目标专注时间
- 时间选项：15分钟、25分钟、30分钟、45分钟、60分钟、自定义
- 默认值：25分钟（番茄钟）

### F3: 专注计时器
- 显示已专注时间（倒计时或正计时）
- 显示目标时间
- 进度条可视化
- 最小化后台运行时保持计时

### F4: 轻音乐播放
- 内置轻音乐列表（至少5首）
- 音乐可选：可开启/关闭
- 音量控制
- 专注结束后自动停止

### F5: 专注完成提示
- 完成后显示赞美文案
- 显示本次专注时长
- 成就徽章（如有）

### F6: 专注历史
- 查看历史专注记录
- 统计：今日专注次数、总时长
- 本周专注趋势

## 4. 数据模型

### 4.1 FocusRecord (专注记录)
```typescript
interface FocusRecord {
  id: string;              // 唯一ID
  startTime: number;       // 开始时间戳
  endTime: number;        // 结束时间戳
  targetDuration: number; // 目标时长(秒)
  actualDuration: number; // 实际时长(秒)
  completed: boolean;     // 是否完成目标
  musicUsed: string;      // 使用的音乐名称
}
```

### 4.2 MusicItem (音乐项)
```typescript
interface MusicItem {
  id: string;
  name: string;
  resource: string;        // 资源路径
  duration: number;       // 时长(秒)
}
```

### 4.3 UserSettings (用户设置)
```typescript
interface UserSettings {
  defaultDuration: number; // 默认专注时长
  lastMusicId: string;    // 上次使用的音乐
  dailyGoal: number;      // 每日目标(分钟)
}
```

## 5. API设计

### 5.1 专注服务 (FocusService)

| 方法 | 说明 | 参数 | 返回 |
|------|------|------|------|
| startFocus(duration) | 开始专注 | duration: number | FocusRecord |
| endFocus() | 结束专注 | - | FocusRecord |
| getFocusing() | 获取当前专注状态 | - | FocusRecord \| null |
| getHistory() | 获取历史记录 | - | FocusRecord[] |
| getTodayStats() | 获取今日统计 | - | {count, totalMinutes} |

### 5.2 音乐服务 (MusicService)

| 方法 | 说明 | 参数 | 返回 |
|------|------|------|------|
| getMusicList() | 获取音乐列表 | - | MusicItem[] |
| play(musicId) | 播放音乐 | musicId: string | void |
| pause() | 暂停 | - | void |
| stop() | 停止 | - | void |
| setVolume(level) | 设置音量 | level: 0-100 | void |

## 6. 技术要求

### 6.1 开发环境
- DevEco Studio 4.0+

- 启动时间 < 2秒
