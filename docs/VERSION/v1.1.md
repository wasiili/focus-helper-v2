# Focus Helper v1.1 - 视觉体验升级

## 版本更新内容

### ✨ 新增功能
- **渐变色主题背景**：从蓝色到紫色的渐变背景，提升视觉舒适度
- **动态粒子效果**：背景添加轻微的浮动粒子，增加沉浸感

### 🎨 UI优化
- 按钮增加阴影和圆角效果
- 字体采用更现代的无衬线体
- 增加微交互动画（按钮点击反馈）

### 🔧 技术改进
- 优化页面切换动画流畅度
- 添加骨架屏加载效果
