### 🧩 组件重构
- 抽取 StatCard 统计卡片组件
- 抽取 DurationPicker 时长选择器组件

- 组件目录结构重组

### 1. 组件划分
- 每个组件单一职责
- Props 接口清晰定义
- 组件内聚性高
- 组件间低耦合

### 2. 可复用性
- DurationPicker 支持多种配置
- AchievementCard 支持自定义样式
- 组件可在其他页面复用
