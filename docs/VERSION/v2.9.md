# Focus Helper v2.9 - 辅助功能与无障碍

## 版本更新内容

### ♿ 无障碍支持
- 屏幕阅读器完整支持
- 所有按钮/图片添加描述标签
- 焦点顺序可自定义
- 高对比度模式

### 👁️ 视觉辅助
- 字体大小可调节（大/中/小）
- 减少动画模式
- 色彩校正选项
- 暗色模式支持

### 🗣️ 语音支持
- 语音播报专注状态
- 语音提醒功能
- 语音控制开始/结束
- 语音查询统计数据
